# DeKalb County CDBG Application Checklist

## Required Documentation

### Property Information
- [ ] Property address and parcel ID
- [ ] Proof of ownership (deed, title insurance policy)
- [ ] Property tax records showing current status
- [ ] Photographs of property (exterior and interior)
- [ ] Site plan or property survey
- [ ] Floor plans showing unit layouts
- [ ] Current rent roll (if property is occupied)

### Project Information
- [ ] Detailed scope of work
- [ ] Line-item budget for rehabilitation work
- [ ] Contractor estimates (minimum of 3 if available)
- [ ] Project timeline with key milestones
- [ ] Energy efficiency improvement details
- [ ] Clean energy components (if applicable)
- [ ] Building permits (or confirmation they can be obtained)

### Financial Information
- [ ] Sources and uses of funds statement
- [ ] Evidence of additional funding sources (if applicable)
- [ ] Operating pro forma (5-year projection)
- [ ] Bank statements or proof of funds for matching portion
- [ ] Most recent property financial statements
- [ ] Property owner's financial statements or tax returns

### Applicant Information
- [ ] Applicant's legal name and contact information
- [ ] Business entity documentation (if applying as LLC, corporation, etc.)
- [ ] DUNS number and SAM registration
- [ ] Resumes of key project team members
- [ ] References from previous rehabilitation projects (if available)
- [ ] Conflict of interest disclosure

### Tenant/Beneficiary Information
- [ ] Current tenant income verification (if property is occupied)
- [ ] Tenant demographic information
- [ ] Relocation plan (if temporary relocation will be necessary)
- [ ] Marketing plan for vacant units
- [ ] Tenant selection criteria
- [ ] Rent schedule (current and proposed)

### Compliance Documentation
- [ ] Environmental review information
- [ ] Historic preservation review (if applicable)
- [ ] Lead-based paint assessment (for pre-1978 buildings)
- [ ] Asbestos assessment (for older buildings)
- [ ] Flood zone determination
- [ ] Section 3 plan (for hiring low-income residents)
- [ ] Fair housing marketing plan

## Application Forms

- [ ] CDBG Application Form (available from DeKalb County)
- [ ] Budget Form
- [ ] Certifications and Assurances Form
- [ ] Environmental Review Form
- [ ] Davis-Bacon Compliance Acknowledgment (if applicable)
- [ ] Section 3 Compliance Form
- [ ] Conflict of Interest Disclosure Form
- [ ] Authorization for Release of Information

## Submission Requirements

- [ ] One original signed application with all attachments
- [ ] Three copies of complete application package
- [ ] Electronic copy on USB drive or CD
- [ ] Submission before the deadline (typically 5:00 PM on the due date)
- [ ] Delivery to correct address:
  ```
  DeKalb County Community Development Department
  330 W. Ponce de Leon Ave, Suite 500
  Decatur, GA 30030
  ```

## Post-Submission Follow-Up

- [ ] Confirmation of receipt from DeKalb County
- [ ] Response to any requests for additional information
- [ ] Attendance at application review meeting (if invited)
- [ ] Site visit preparation (if scheduled by county staff)
- [ ] Award notification tracking

## If Awarded

- [ ] Grant agreement review and execution
- [ ] Environmental clearance completion
- [ ] Contractor selection process documentation
- [ ] Pre-construction meeting with county staff
- [ ] Construction monitoring documentation
- [ ] Draw request preparation
- [ ] Compliance reporting setup
