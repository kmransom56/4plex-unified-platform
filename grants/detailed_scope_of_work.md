# Detailed Scope of Work
## 2072 E Roxboro Rd NE, Brookhaven, GA 30324

This document provides a comprehensive scope of work for the rehabilitation and energy efficiency upgrades of the multifamily property located at 2072 E Roxboro Rd NE. The scope is organized by building component and includes specific requirements to meet grant program standards.

## 1. Site Work

### 1.1 Drainage Improvements
- Evaluate existing site drainage patterns
- Install/repair gutters and downspouts with extensions directing water away from foundation
- Regrade soil around foundation to ensure positive drainage (min. 6" drop in 10')
- Install French drain system in areas with persistent drainage issues

### 1.2 Walkways and Access
- Repair cracked/uneven concrete walkways
- Ensure ADA-compliant access to at least one unit entrance
- Install energy-efficient exterior lighting with motion sensors
- Repair/replace exterior stairs as needed with code-compliant materials

### 1.3 Landscaping
- Trim vegetation away from building (min. 12" clearance)
- Remove any trees threatening structure
- Install drought-resistant native plants
- Apply mulch around foundation plantings (min. 2" depth)

## 2. Building Exterior

### 2.1 Roof System
- Complete roof inspection and assessment
- Replace roof covering with ENERGY STAR rated materials
- Install proper attic ventilation (ridge and soffit vents)
- Replace damaged fascia and soffit boards
- Install leaf guards on gutters
- Inspect and repair chimney as needed

### 2.2 Exterior Walls
- Repair/replace damaged siding
- Pressure wash and prep all exterior surfaces
- Apply high-quality exterior paint with 15-year warranty
- Caulk and seal all exterior joints, penetrations, and openings
- Repair/replace damaged trim and decorative elements

### 2.3 Windows and Doors
- Replace all windows with ENERGY STAR rated double-pane windows
  - U-factor: ≤ 0.30
  - Solar Heat Gain Coefficient (SHGC): ≤ 0.25
- Install proper flashing and weatherstripping
- Replace exterior doors with ENERGY STAR rated insulated doors
  - U-factor: ≤ 0.30
- Install new door hardware with deadbolts
- Install peepholes at appropriate height
- Replace/repair storm doors as needed

### 2.4 Foundation
- Inspect foundation for cracks and damage
- Repair foundation cracks using appropriate methods
- Install/repair vapor barrier in crawl space
- Install foundation vents as needed for proper ventilation
- Treat for termites and other wood-destroying organisms

## 3. Building Systems

### 3.1 HVAC System
- Remove and properly dispose of existing HVAC equipment
- Install new high-efficiency heat pump system
  - Minimum 16 SEER
  - Minimum 9 HSPF
  - ENERGY STAR certified
- Size equipment according to Manual J calculations
- Install programmable or smart thermostats
- Clean and seal all ductwork
  - Perform duct blaster test to verify leakage < 10%
- Install fresh air intake with filtration
- Balance system for optimal performance

### 3.2 Electrical System
- Upgrade electrical service to 200 amps if needed
- Replace outdated electrical panel
- Install AFCI/GFCI protection as required by code
- Replace outdated/damaged wiring
- Install smoke and carbon monoxide detectors
  - Hardwired with battery backup
  - Interconnected between units
- Replace all lighting fixtures with ENERGY STAR LED fixtures
- Install ceiling fans in living rooms and bedrooms
- Upgrade exterior lighting with photocell/motion sensors

### 3.3 Plumbing System
- Replace galvanized/outdated water supply lines
- Install water-saving fixtures throughout
  - Toilets: 1.28 gpf or less
  - Showerheads: 1.5 gpm or less
  - Faucets: 1.0 gpm or less
- Replace water heaters with high-efficiency models
  - Heat pump water heaters where feasible (min. EF 2.0)
  - Or tankless water heaters (min. EF 0.90)
- Insulate all hot water pipes
- Install shut-off valves at all fixtures
- Repair/replace damaged drain lines
- Install backflow preventers as required

### 3.4 Solar PV System (Phase 2)
- Install roof-mounted solar PV system
  - Minimum 5kW per unit
  - High-efficiency panels (min. 19% efficiency)
- Install inverter and monitoring system
- Connect to electrical panel with appropriate disconnects
- Obtain necessary permits and utility interconnection agreement
- Configure for net metering

## 4. Building Envelope

### 4.1 Attic Insulation
- Remove existing insulation if damaged or inadequate
- Air seal all penetrations in attic floor
  - Around chimney/flues (with appropriate materials)
  - Electrical and plumbing penetrations
  - Top plates of walls
  - Attic access hatch
- Install baffles at eaves to maintain ventilation
- Install R-49 blown cellulose or fiberglass insulation
- Insulate and weatherstrip attic access hatch

### 4.2 Wall Insulation
- Assess existing wall insulation using thermal imaging
- Dense-pack wall cavities with cellulose insulation
- Install continuous exterior insulation during siding replacement (if applicable)
- Seal all penetrations through exterior walls

### 4.3 Floor Insulation
- For units over crawlspace:
  - Install minimum R-19 floor insulation
  - Secure with proper fasteners
  - Install ground moisture barrier (6-mil polyethylene)
- For slab-on-grade:
  - Insulate slab perimeter where accessible

### 4.4 Air Sealing
- Perform blower door test to identify leakage areas
- Seal all identified leaks using appropriate materials
- Target final building leakage: < 5 ACH50
- Seal around all windows and doors
- Seal all penetrations in building envelope
- Weatherstrip all exterior doors

## 5. Unit Interiors

### 5.1 Kitchens
- Install ENERGY STAR certified appliances
  - Refrigerator
  - Dishwasher
  - Range/oven
- Install new energy-efficient lighting
- Replace countertops and cabinets as needed
- Install water-saving kitchen faucet
- Repair/replace flooring with durable, low-VOC materials
- Paint walls and ceilings with low-VOC paint

### 5.2 Bathrooms
- Install water-saving fixtures
- Replace vanities and toilets as needed
- Install GFCI outlets
- Install energy-efficient ventilation fans vented to exterior
  - ENERGY STAR rated
  - Humidity sensing or timer controls
- Repair/replace flooring with water-resistant materials
- Install grab bars or blocking for future installation
- Repair/replace tub/shower surrounds as needed

### 5.3 Living Areas and Bedrooms
- Repair/replace damaged drywall
- Paint all walls and ceilings with low-VOC paint
- Replace damaged flooring with durable, low-VOC materials
- Install energy-efficient lighting fixtures
- Install ceiling fans in all bedrooms and living rooms

### 5.4 Closets and Storage
- Repair/replace closet shelving and rods
- Install energy-efficient lighting
- Repair/replace doors and hardware

## 6. Health and Safety

### 6.1 Lead-Based Paint
- Conduct lead-based paint assessment
- Implement lead-safe work practices
- Abate or encapsulate lead paint as required
- Clearance testing after work completion

### 6.2 Asbestos
- Test suspected asbestos-containing materials
- Abate or encapsulate as required
- Follow all regulations for handling and disposal

### 6.3 Mold and Moisture
- Identify and address sources of moisture
- Remediate any existing mold following proper protocols
- Install humidity monitors in each unit
- Ensure proper ventilation in bathrooms and kitchens

### 6.4 Pest Control
- Inspect for and treat any pest infestations
- Seal entry points for pests
- Install pest-resistant features (door sweeps, screens, etc.)

### 6.5 Fire Safety
- Install interconnected smoke and CO detectors
- Ensure proper fire separation between units
- Verify/install fire-rated doors as required
- Ensure clear egress paths from all bedrooms

## 7. Accessibility Improvements

### 7.1 Common Areas
- Ensure accessible route to building entrance
- Install ramp if needed (1:12 slope maximum)
- Install handrails at appropriate height
- Ensure adequate lighting along accessible route

### 7.2 Unit Accessibility (for at least one unit)
- Widen doorways to minimum 32" clear width
- Install lever-style door handles
- Install accessible switches and outlets
- Create zero-step entrance
- Install grab bars in bathroom
- Create accessible bathroom with roll-in shower or transfer space
- Ensure kitchen accessibility (counter heights, sink access, etc.)

## 8. Energy Efficiency Testing and Verification

### 8.1 Pre-Rehabilitation Testing
- Blower door test to measure air leakage
- Duct blaster test to measure duct leakage
- Infrared thermal imaging to identify insulation gaps
- Combustion safety testing for existing equipment
- Record baseline energy usage data

### 8.2 Post-Rehabilitation Testing
- Repeat blower door test (target: 50% reduction in air leakage)
- Repeat duct blaster test (target: <10% leakage)
- Verify proper HVAC operation and balance
- Verify proper ventilation operation
- Combustion safety testing for new equipment
- Verify proper operation of all new systems

## 9. Project Management Requirements

### 9.1 Documentation
- Detailed before and after photos of all work areas
- Product specifications and warranties for all installed equipment
- Building permits and inspection records
- Lead and asbestos testing/clearance reports
- Energy efficiency testing results
- Manufacturer's installation instructions for all equipment

### 9.2 Quality Control
- Daily site inspections during construction
- Third-party verification of energy efficiency measures
- Inspection at mechanical rough-in stage
- Inspection prior to drywall installation
- Final inspection with detailed punch list
- Homeowner/tenant orientation for all new systems

### 9.3 Warranties
- Minimum 1-year warranty on all labor
- Pass through all manufacturer warranties to owner
- Specific system warranties:
  - Roof: Minimum 25-year materials warranty
  - HVAC: Minimum 10-year parts, 2-year labor
  - Appliances: Minimum 1-year parts and labor
  - Windows/Doors: Minimum 20-year warranty
  - Solar PV System: Minimum 25-year panel warranty, 10-year inverter warranty

- Total project duration not to exceed 120 calendar days
- Phased approach to minimize tenant disruption

- Coordination with tenant relocation schedule
- Weather contingency days included in schedule
- Critical path identification and management
